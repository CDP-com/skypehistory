Run

The Skype History App places a folder called SkypeChatHistory on your computer's Desktop. History files are placed inside the folder as text files. Double click them to read them.

The Skype History App is used on Microsoft Windows operating systems.

The Skype application, version 5.6.0.105 or greater, must be installed on the local machine to use this App.

The App scans existing Skype chat history and creates text files for each chat group in a folder on the computer's Desktop.

Script Source Code

Q: What does this app do?
A: This app adds a folder to your desktop which will contain text files of your Skype chats.

Q: Who is this app for?
A: This app is for everyone with a Windows machine and who has Skype.
